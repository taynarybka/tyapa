Тяпа 1.2

Папки:
- computer-only — локальная версия для компьютера. Ручной ввод шагов виден сразу.
- wifi-netlify — версия для Netlify и локального WiFi-сервера. netlify.toml настроен на эту папку.

Что входит:
- старт карты 7 x 7;
- обучение с мягкой подсказкой идти к дальнему углу карты;
- шаги -> тяптяпы, статистика за последние 7 дней;
- полет, посев тропинок, скорлупка;
- дневник и полочка уже видны в скорлупке, но пока недоступны и некликабельны;
- персонажи, портреты и их ассеты исключены из сборки 1.2.

Шаги:
- computer-only: шаги вводятся вручную, кнопка "Обновить" начисляет разницу в тяптяпы.
- wifi-netlify: кнопка "Подключить Здоровье" ищет JS-мост window.tyapaSteps / window.TyapaSteps. Если моста нет, включается тестовое подтверждение и демо-данные за 7 дней для проверки UX и конвертации на телефоне.

Мобильные фиксы:
- первый экран с доступом к шагам ужат под 320/360/375/390 px;
- карта не должна уходить под нижний край на 320 x 568;
- скорлупка на 320 x 568 переведена в компактную сетку без наложения кнопок;
- тап по "Посеять" отделен от "Полет" на маленьких экранах.
